¿Qué son las condicionales?

En JavaScript, las condicionales permiten que el programa ejecute diferentes bloques de código dependiendo de si una condición es verdadera o falsa. Esto ayuda a que el código sea más flexible y responda a diferentes situaciones.

Ejemplo de declaraciones


// Ejemplo de declaraciones
let x = 5;  // Declaración de variable
console.log(x);  // Llamada a función
x++;  // Operación

if / else


// Ejemplo básico
let edad = 18;

if (edad >= 18) {
    console.log("Eres mayor de edad");
} else {
    console.log("Eres menor de edad");
}

if...else if...else


let nota = 85;

if (nota >= 90) {
    console.log("Excelente");
} else if (nota >= 70) {
    console.log("Aprobado");
} else {
    console.log("Reprobado");
}

Operador ternario


let hora = 20;
let saludo = (hora < 18) ? "Buenas tardes" : "Buenas noches";
console.log(saludo);

switch


let dia = 3;

switch (dia) {
    case 1:
        console.log("Lunes");
        break;
    case 2:
        console.log("Martes");
        break;
    case 3:
        console.log("Miércoles");
        break;
    default:
        console.log("Día no válido");
}
