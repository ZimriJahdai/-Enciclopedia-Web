Funciones en JavaScript

Las funciones son bloques de código reutilizables que ejecutan una tarea específica. Permiten organizar mejor el código, evitar repeticiones y hacerlo más legible.

1. Función Declarada

Son las funciones definidas con la palabra clave function y pueden ser llamadas antes de su definición debido al "hoisting".

// Ejemplo de función declarada
function saludar(nombre) {
    return "Hola, " + nombre + "!";
}
console.log(saludar("Juan"));

2. Función Expresada

Se asigna una función a una variable. No se pueden usar antes de ser definidas.

// Ejemplo de función expresada
const sumar = function(a, b) {
    return a + b;
};
console.log(sumar(5, 3));

3. Función de Flecha

Más compactas y no vinculan su propio this. Son útiles para callbacks y funciones cortas.

// Ejemplo de función de flecha
const multiplicar = (a, b) => a * b;
console.log(multiplicar(4, 2));

4. Parámetros por Defecto

Permiten asignar un valor por defecto si el argumento no se proporciona.


function saludarPorDefecto(nombre = "Visitante") {
    return "Hola, " + nombre + "!";
}
console.log(saludarPorDefecto());

5. Funciones Anónimas

Son funciones sin nombre, usadas frecuentemente como argumentos de otras funciones.


setTimeout(function() {
    console.log("Esto se ejecuta después de 2 segundos");
}, 2000);

6. Funciones como Argumentos

En JavaScript las funciones son ciudadanos de primera clase, lo que significa que pueden ser pasadas como parámetros.


function operar(a, b, operacion) {
    return operacion(a, b);
}
console.log(operar(5, 3, (x, y) => x - y));

7. Funciones Autoejecutables (IIFE)

Se ejecutan inmediatamente después de su creación.


(function() {
    console.log("Soy una función autoejecutable");
})();
Consejo: Divide tu código en funciones pequeñas y específicas para mejorar su mantenimiento y evitar errores.
