¿Qué es JavaScript?

JavaScript es un lenguaje de programación interpretado, dialecto del estándar ECMAScript. Se define como orientado a objetos, basado en prototipos, imperativo, débilmente tipado y dinámico.

Dato clave

JavaScript no tiene relación con Java, a pesar del nombre. El nombre fue una decisión de marketing cuando se lanzó.

JavaScript se utiliza principalmente del lado del cliente (frontend), implementado como parte de un navegador web permitiendo mejoras en la interfaz de usuario y páginas web dinámicas.

Historia de JavaScript

JavaScript fue creado en 1995 por Brendan Eich mientras trabajaba en Netscape Communications Corporation. Fue desarrollado en solo 10 días bajo el nombre de Mocha, luego LiveScript, y finalmente JavaScript.

Año Hito
1995 Nace JavaScript en Netscape
1997 Se estandariza como ECMAScript
2005 AJAX populariza JavaScript
2009 Node.js permite JS en el servidor
2015 ES6 (gran actualización del lenguaje)

¿Cómo funciona JavaScript?

JavaScript es un lenguaje interpretado que se ejecuta en el navegador del usuario. Cuando una página web carga, el navegador:

  1. Descarga el código JavaScript
  2. Lo analiza (parsing)
  3. Lo compila Just-In-Time (JIT)
  4. Lo ejecuta
Importante

JavaScript es single-threaded, lo que significa que solo puede ejecutar una operación a la vez. Sin embargo, usa un modelo asíncrono y no bloqueante con el event loop.

Tu primer código JavaScript

El clásico "Hola Mundo" en JavaScript se puede escribir de varias formas:

// Opción 1: Mostrar en consola
console.log("Hola Mundo");

// Opción 2: Mostrar en el documento
document.write("Hola Mundo");

// Opción 3: Mostrar en un alert
alert("Hola Mundo");
Práctica recomendada

Para desarrollo moderno, la mejor opción es console.log() para depuración y manipulación del DOM para mostrar contenido al usuario.

¿Dónde se usa JavaScript hoy?

JavaScript ha evolucionado mucho desde sus inicios y ahora se usa en:

Frontend Web

Frameworks como React, Angular y Vue.js

Backend

Con Node.js, Express, NestJS

Aplicaciones móviles

React Native, Ionic, NativeScript

Aplicaciones de escritorio

Electron, NW.js
